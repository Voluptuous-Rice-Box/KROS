# OS

## Dependencies
```bash
sudo apt install nasm
sudo apt install xorriso
sudo apt install gcc-multilib g++-multilib
```

If you are running on EFI:
```bash
sudo apt install grub-pc-bin
```

## Building
```bash
make
```
or
```
make iso
```
